# Building an AI powered ChatBot with Pieces OS Client Python SDK

## Getting Started

1. Prerequisites: 
   
   - python 3.9.0+ version

2. Install the dependencies:

   ```
   pip install -r requirements.txt
   ```

3. Run the app

   ```python
   streamlit run pieces_streamlit.py
   ```

## Dependencies

- pieces_os_client
- streamlit
